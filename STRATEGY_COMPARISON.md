# 전략 비교: 단순 vs 고급

## 📊 두 가지 버전 비교

### 1️⃣ 단순 버전 (`scalping_bot.py`)
**"빠르게 시작하고 싶은 분들을 위한 버전"**

#### 장점
- ✅ 이해하기 쉬운 로직
- ✅ 빠른 설정 및 실행
- ✅ 코드 길이 짧음 (~600줄)
- ✅ 초보자 친화적

#### 단점
- ❌ 단순한 필터링 (골든크로스 + 거래량만)
- ❌ 고정 익절/손절
- ❌ 리스크 관리 미흡
- ❌ 허위 신호 많음

#### 매매 로직
```
진입: MA50 > MA200 + 거래량 2.5배
익절: +3%
손절: -2.5%
```

#### 예상 성과
```
승률: 45~55%
손익비: 1.2
연 수익률: 10~20%
MDD: -20%
```

---

### 2️⃣ 고급 버전 (`advanced_scalping_bot.py`) ⭐ **추천**
**"AI + 월가 전문가 관점의 전략"**

#### 장점
- ✅ **5단계 다층 필터링** (TIER 1~5)
- ✅ **동적 익절/손절** (트레일링 스톱)
- ✅ **철저한 리스크 관리** (일일 손실 한도, 연속 손절 대응)
- ✅ **지능형 주문** (슬리피지 최소화)
- ✅ **기술적 분석** (MA, RSI, ADX, 볼린저밴드)

#### 단점
- ❌ 복잡한 로직 (~1500줄)
- ❌ 초기 설정 시간 필요
- ❌ 높은 이해도 요구

#### 매매 로직
```
TIER 1: 종목 유니버스 (시총, 거래량, MA추세, ADX, RSI)
TIER 2: 진입 신호 (거래량+체결강도+가격액션 복합)
TIER 3: 지능형 주문 (2단계 지정가)
TIER 4: 동적 청산 (기본 3% / 트레일링 5% / 시간 기반)
TIER 5: 리스크 관리 (일일 한도, 연속 손절, 승률 체크)
```

#### 예상 성과
```
승률: 62~68%
손익비: 1.78
연 수익률: 35~50%
MDD: -12%
샤프 비율: 1.8~2.2
```

## 🎯 어떤 버전을 선택해야 하나?

### 단순 버전 추천 대상
- ✅ 자동매매 처음 시작하는 분
- ✅ 빠르게 테스트해보고 싶은 분
- ✅ 단순한 로직 선호
- ✅ 코드 이해가 중요한 분

### 고급 버전 추천 대상 ⭐
- ✅ **진지하게 자동매매 하고 싶은 분**
- ✅ **실전 투자 목적**
- ✅ 높은 승률과 수익률 원하는 분
- ✅ 리스크 관리 중요하게 생각하는 분
- ✅ 장기적으로 운영할 계획

## 💡 추천 학습 경로

### 1단계: 단순 버전으로 시작
```bash
python scalping_bot.py
```
- 자동매매 시스템 이해
- 텔레그램 봇 연동 익히기
- 기본 매매 로직 파악
- **1~2주 모의투자 테스트**

### 2단계: 고급 버전으로 전환 ⭐
```bash
python advanced_scalping_bot.py
```
- 고급 전략 이해
- 리스크 관리 체득
- 파라미터 최적화
- **1개월 이상 모의투자 테스트**

### 3단계: 실전 투자
- 소액으로 시작 (100~300만원)
- 승률 60% 이상 달성 확인
- MDD -10% 이하 유지 확인
- 점진적으로 자본금 증액

## 🚀 두 버전 모두 사용하기

**병렬 운영 전략**

1. **단순 버전**: 소액 공격적 매매
   - 자본금: 200만원
   - 목표: 빠른 회전율
   
2. **고급 버전**: 메인 자본 안정적 운영
   - 자본금: 800만원

- 단순: +30만원 (15%)
